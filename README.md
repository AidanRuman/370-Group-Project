## Overview
An interactive mobile app that helps users explore classical history, mythology, and philosophy through an engaging, map-based experience.

## Core Idea
- Combats the lack of accessible knowledge in classics and philosophy  
- Uses a Mediterranean map (focused on Greece and Italy) to connect ideas, people, and events to real locations  

## Key Features
- **Map Tab:** Displays important historical events and philosophers with location-based markers  
- **Philosophy Tab:** Highlights major thinkers and their ideas  
- **Classics Tab:** Covers history and mythology  

## Technical Approach
- Built with a tab-based navigation system and dynamic detail pages  
- Uses Supabase for storing content (text and images)  
- Fetches data dynamically with TanStack Query  

## Additional Functionality
- Notifications for important dates (e.g., birthdays, historical anniversaries)  
- Expandable content pages for deeper learning  

## Team & Workflow
- Developed collaboratively with defined roles and GitHub-based workflow  
- Feature-based branching for Maps, Classics, and Philosophy
